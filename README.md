### React_Shop

React를 활용해 개발한 경매플랫폼 컨셉의 쇼핑몰입니다.(개발 진행중)

링크:https://react-redux-yungukplan.web.app/

반응형: mediaquery를 활용하여 추후 반응형 작업 예정

기능: 로그인, 회원가입, 로그아웃, 아이템 리스트, 아이템 상세보기, 카트 추가하기, 카트 리스트
